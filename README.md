# MiscellaneousScripts

A collection of random scripts written to solve specific problems encountered in day-to-day life, work, hobbies, and personal projects.

Most scripts in this repository were created for a single task and may never be used again. Rather than letting them disappear into forgotten folders, they are collected here for future reference. Occasionally an old script turns out to be useful months or years later, either as a complete solution or as a starting point for something new.

## What You’ll Find Here

Examples of the kinds of things that may appear in this repository:

* Data conversion and cleanup utilities
* File and folder management scripts
* Metadata normalization tools
* CSV import/export helpers
* One-time migration scripts
* Personal automation tools
* Research and data-processing utilities
* Miscellaneous experiments that solved a real problem

## Philosophy

Most scripts are designed to be:

* Simple
* Self-contained
* Easy to understand
* Easy to modify
* Useful enough to save time

Code quality, architecture, and reusability are secondary to solving the problem at hand.

## Structure

Scripts may be organized into folders when a common theme emerges, but many will remain standalone files.

No guarantees are made regarding consistency, coding style, maintenance, or long-term support.

## Usage

Before running any script:

1. Read the code.
2. Verify any paths, filenames, or configuration values.
3. Back up any data the script may modify.
4. Test on a small sample whenever possible.
